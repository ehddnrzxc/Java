소스코드 작성 -> 소스코드는 클래스 내부에 있음 -> 파일명과 클래스명은 동일
-> 컴파일(소스 코드를 이진 데이터로 변경) -> 컴파일 성공 시 class 파일 생성 
-> 실행(.java를 컴파일 후 class파일 생성 후 실행)

컴파일: javac.exe

실행: java.exe


자바프로젝트
 ㄴ src
   ㄴ package 
   ㄴ HellowWorld.java
 ㄴ bin
   ㄴ package
   ㄴ HellowWorld.class

패키지에 여러 클래스를 보관(같은 이름의 클래스도 다른 패키지로 보관하면 괜찮다.)
